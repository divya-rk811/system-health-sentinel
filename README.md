#  System Health Sentinel

A lightweight Python utility designed to monitor hardware performance and analyze local storage health. This project demonstrates proficiency in **OS Interaction**, **Resource Management**, and **File System Analysis**.

##  Key Features
- **Performance Vitals:** Tracks real-time CPU and RAM utilization.
- **Automated Alerts:** Triggers warnings when system thresholds (e.g., >80% CPU) are exceeded.
- **Storage Audit:** Scans directories to identify large files that impact system performance.
- **Cross-Platform:** Designed to function on Windows, macOS, and Linux.

##  Tech Stack
- **Language:** Python 3.x
- **Libraries:** Psutil (Process & System Utilities), Platform, OS
